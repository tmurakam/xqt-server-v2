X/Qt Server Help

Qtopia 上で動作する X サーバです。

本サーバには Xlib やフォントなどは入ってないので、別途用意してください。 なお、フォントは /opt/QtPalmtop/lib/X11/fonts に入れてください。

Xクライアントをリモート実行する場合は、xhost や xauth などを使うか、Xqt のコマンドライン引数に -ac をつける必要があります。

フルスクリーンモードへの切替はメニューから行えます。

キー操作について

Menu キーでポップアップメニュー表示、フルスクリーンモード解除を行います。

特殊キーは以下のように割り当てられています。
  • Escape : Cancel
  • Control : Address
  • Alt : Home

Fn+{Q,O,P,A,S,D,Z,X,C,N,M} はそのまま Ctrl キーとの組み合わせとして働きますが、 Address キーを使ったほうがいいでしょう、、、
` は Fn + Shift + '-' で入ります。

マウス操作について

  • そのままタップ : 左クリック
  • Fn キーを押したままタップ : 右クリック
  • Fn+Shift キーを押したままタップ: 中クリック
