X server running on Qtopia.

Key bindings

Menu key: Popup menu, leave full screen mode.

Special keys are binded as:
  • Escape : Cancel
  • Control : Address
  • Alt : Home


'`' are binded with Fn + Shift + '-'.

Mouse buttons

  • Tap : Left click
  • Fn + Tap : Right click
  • Fn + Shift + Tap : Center click
